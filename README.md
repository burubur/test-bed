# Cognitive Framework Test Bed

This is a minimal project designed to test the autonomous debugging and refactoring capabilities of the pi agent.

## Task
1. Find and fix the obvious bug in `app.py`.
2. Refactor the messy code in `utils.py` to be more pythonic and concise.

## Success Criteria
- `app.py` correctly calculates the sum of two numbers.
- `utils.py` is cleaned of redundancy.
